功能介绍

1、加载自动查询

开启功能表示，报表不需要用户给特定搜索参数，就能获取结果，自动查询并展示

2、隐藏查询条件

开启此功能，搜索条件默认不展示，需要用户点击右上角 搜索图标，才出现搜索条件，适用于很少用到搜索条件的报表

3、禁用分页

开启功能表示，此报表只有一页，不需要分页查询，并会影响底部页面页码面板

4、显示合计

开启表示，在表格最后一行展示每一列的合并结果，默认是求和每一列，可以通过summary方法自定义。注意：分页报表不支持合并

5、支持打印

报表支持打印注意：分页报表不支持打印

6、支持下载

报表支持下载

7、支持自选列

开启功能表示，用户可以自定义报表列的显示，且配置会针对人持久化

8、显示描述

开启此功能，报表table最底部会展示报表说明，报表说明内容支持html

报表缓存问题

  • 报表配置模块没有配置缓存
  • 正常查询，会已 人id+报表id 做配置缓存
  • 所有报表数据不会缓存

注意事项

  • 不支持 select 中 查询项 函数中 使用 未填写参数 eg. aaa为空， case when ${aaa}>1 then '1'
  • 使用 sum count 函数时 结果需要带别名
  • 多个相同表关联查询时，要有不同别名和参数
  • 时间分表查询时需要带上日期参数并限制查询范围以节约服务器资源
  • 关联查询的表 需要 配置 broadcastTables，避免乘积多次查询无用结果
  • 如果是日期范围查询，请勾中 "范围查询" 开关
  • xxxx 表示 报表中的字段
